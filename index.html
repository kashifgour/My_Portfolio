Projects

Campus Queries App

The Campus Queries app assists prospective students in accessing accurate information about DIT University admissions and related queries. It features a custom dataset with FAQs and relevant links for detailed information, along with search functionality for real-time responses. The app allows easy navigation to topics such as fees, placements, and campus facilities, all within a user-friendly interface, ensuring smooth interaction for students seeking admission details.

Tech : Python, NLP, LangChain, Gemini 1.5 Flash, Google Gemini API, Streamlit.
Medical Assistant Chatbot

The medical assistant chatbot, built using Gemini 1.5 Flash and Flask, is designed to assist users with healthcare-related queries. It leverages Retrieval-Augmented Generation (RAG) to deliver accurate, context-aware responses and offers a downloadable heart attack guide in PDF format. With real-time response capabilities and a user-friendly interface, the chatbot ensures users can easily access reliable and relevant medical information.

Tech : Gemini 1.5 Flash, Flask, HTML, CSS, LangChain, RAG, PDF integration, Python, Render.
Flight Prices Prediction System

The flight prices prediction system is designed to accurately forecast flight prices by utilizing AWS SageMaker for the complete workflow. This encompasses data collection, preprocessing, model training, evaluation, and deployment. By harnessing AWS SageMaker's capabilities, I effectively streamlined the process and ensured efficient data management, resulting in a robust model for predicting flight prices.



Tech : Python, Machine Learning, XGBoost, AWS SageMaker, Streamlit.
